# Instagram Song Feature 🎵

A Flask-based Instagram-style web application where users can create posts with videos and music.

## Features

- User signup and login
- Upload video posts
- Add songs to posts
- Change song without deleting post
- Like system
- Comment system
- User profile page
- Follow system

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

python app.py

Open:


http://127.0.0.1:5000
